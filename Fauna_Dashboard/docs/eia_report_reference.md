| 분류군 | 세부 조사 내용 |
|---|---|
| **육상식물** | 관속식물, 식물구계학적 특정종, 귀화식물, 생태계교란 생물(식물), 희귀식물, 특산식물, 법정보호종 |

| **육상동물 > 육상곤충류** | 출현 종 및 개체수, 생태계교란 생물, 법정보호종 |

#### 조사 시기 (조사 횟수)
- **표준**: 계절별 4회 이상 (봄/여름/가을/겨울)
- 법정보호종(맹꽁이 등) 특별조사: 번식기 기준 월 1회 추가 (5~8월)
- **조사표 필수 기재 항목**: 조사 구분(차수), 조사 일자, 조사 분류군

#### 조사 방법 (분류군별)
| 분류군 | 방법 |
|---|---|
| 식물상 | 현지답사(라인트랜섹트), 동정(국가생물종지식정보시스템 기준) |
| 포유류 | 무인센서카메라, 핏폴트랩, 청음생물탐지장치(Song-meter), 선조사 |
| 조류 | 정점조사법(Point Count), Road-side census (시속 1.5~2.0km) |
| 양서·파충류 | 라인트랜섹트, 핏폴트랩 |
| 육상곤충 | 말레이즈 트랩, 버킷라이트 트랩, 핏폴트랩 |
| 어류 | 투망법, 족대법 |
| 저서성 무척추동물 | D-frame net, Surber sampler |

### 3-3. 군집분석 지수 (Community Analysis Indices) ⭐ 필수

> **환경부고시 제2023-72호 기준 — 대시보드 계산 기능 필수 구현 항목**

| 지수 | 기호 | 산출식 | 저자·연도 |
|---|---|---|---|
| **우점도 지수** | DI | `DI = (N1 + N2) / N` | McNaughton (1967) |
| **다양도 지수** | H' | `H' = -∑(Ni/N × log₂(Ni/N))` | Shannon-Weaver (1949) |
| **종풍부도 지수** | R1 | `R1 = (S-1) / ln(N)` | Margalef (1958) |
| **균등도 지수** | J' | `J' = H' / log₂S` | Pielou (1975) |

- `N`: 총 개체수  
- `N1, N2`: 제1, 2 우점종 개체수  
- `S`: 총 종수  
- `Ni`: i번째 종의 개체수  

**수질평가 지수 (TESB — 저서성 무척추동물 전용)**
- 한국의 저서성 대형무척추동물 생태점수 기준 (공 등, 2018)
- `Qi`: 환경질점수 (1~5등급)

## 4. 나. 영향예측 (Impact Prediction Section)

### 필수 기재 사항
1. **공사 시 영향**: 지형 훼손, 비산먼지, 소음·진동, 서식지 단절
2. **운영 시 영향**: 도시화로 인한 생태계 변화, 불투수층 증가
3. **직접 영향**: 훼손 면적, 영향받는 법정보호종 개체 수 및 서식지 면적
4. **간접 영향**: 인근 녹지·생태축과의 단절 여부

## 8. 웹 대시보드 필수 탑재 기능 (Gap Analysis)

현재 `Fauna_Dashboard`와 이상적인 보고서 요구사항 사이의 갭 분석 및 구현 우선순위:

| 항목 | 현재 대시보드 수준 | 최종 보고서 요구사항 | 우선순위 |
|---|---|---|---|
| **군집분석 지수 계산기** | 없음 | H', DI, R1, J' 자동 연산 및 차트화 | 🔴 높음 |
| **조사 시기/차수 비교** | 없음 | 4계절 4차 조사 기록 관리 및 비교 | 🔴 높음 |
| **광역조사 체크리스트** | 없음 | 18개 항목 ◎/× 입력 및 DB 저장 | 🟡 중간 |
| **법정보호종 차수별 매트릭스** | 단순 플래그 지정만 | 분류군별 × 차수별 상세 매트릭스 표 | 🔴 높음 |
| **식생보전등급 및 등급제 관리** | 없음 | 생태·자연도 등급, 식생보전등급 기입 필드 | 🟡 중간 |
| **귀화율/도시화지수 산출** | 없음 | `귀화식물 수 / 전체 분류군 수 × 100` 자동 계산 | 🟡 중간 |
| **TESB 생태 수질 평가지수** | 없음 | 저서성 대형무척추동물 생태점수 자동 연산 | 🟠 낮음 |
