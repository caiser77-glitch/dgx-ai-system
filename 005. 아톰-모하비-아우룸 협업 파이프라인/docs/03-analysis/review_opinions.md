## 🔍 핵심 검토 의견 (Analysis)

### 1. 동기화 방식의 정합성 및 충돌 예방 (Rsync vs Git)
* **현황 및 문제점:** 단순 Rsync 기반의 양방향 동기화는 모하비(맥북)에서 초안을 집필하는 도중 아톰 서버가 동기화를 수행하면 파일이 덮어씌워지거나 버전 충돌(Conflict)이 날 수 있습니다. 또한 맥북의 특성상 슬립 모드(Sleep) 진입 시 동기화 단절 및 세션 끊김 현상이 상시 발생할 수 있습니다.
* **검토 의견:** 파일의 상태 변경이 일어날 때 충돌을 완벽히 방지하고, 단절된 세션에서도 안전하게 이력을 보관할 수 있는 구조가 필요합니다.

### 2. 파일명 유니코드 자모분리 (NFC / NFD) 방지 기법
* **현황 및 문제점:** macOS는 파일명 인코딩에 NFD(자모분리: `ᄀᆞᆫ`) 방식을 쓰고, 리눅스 및 윈도우는 NFC(자모결합: `간`) 방식을 씁니다. Rsync를 그대로 실행할 경우, 파일명 인코딩 불일치로 인하여 리눅스나 윈도우에서 RAG 색인 시 문자열 검색이 실패하거나 중복 파일로 오인할 가능성이 매우 높습니다.
* **검토 의견:** 동기화 전송 계층 및 파이썬 로직 전반에서 NFC 변환 처리가 철저하게 가드(Guard)되어야 합니다.

### 3. 피드백 DB (`aurum_nas_rules.db`) 공유 범위 확대
* **현황 및 문제점:** 사용자의 피드백을 통해 갱신되는 분류 및 요약 교정용 SQLite DB가 아톰 서버 로컬에만 한정되어 있습니다. 모하비와 아우룸도 요약 보고서를 파싱하고 한글/PDF 배포를 위해 이 규칙을 참조해야 할 필요가 있습니다.
* **검토 의견:** 피드백 DB 파일도 동기화 대상에 포함하거나, 맥북/아우룸 측에서 오프라인 캐시 형태로 해당 DB를 주기적으로 전달받는 로직이 설계에 포함되어야 합니다.

### 4. 상태 트래킹 스케줄러의 I/O 효율성 및 예외 처리
* **현황 및 문제점:** 마크다운 파일들의 Frontmatter 내 `status` 키값을 매번 폴링해서 읽어오는 방식은 파일 수가 늘어남에 따라 디스크 I/O 오버헤드를 발생시킵니다. 또한 상태 전이 오류나 변환 실패 등 예외 발생 시의 에러 관리 방안이 명시되어 있지 않습니다.
* **검토 의견:** 상태 불일치 및 예외 발생 시 관리자에게 즉시 Telegram/Slack 경고를 보낼 수 있는 예외 처리 상태(`status: error_failed`) 설계가 추가되어야 합니다.

## 🚀 개선 실행 방향 (Action Plan)

개발 일정을 고려하여 즉각적으로 반영할 수 있는 구체적인 개선 방향을 제시합니다.

### 💡 개선안 1. Rsync 전송 시 NFC 변환 및 디렉토리 구조 분할
1. **Rsync 옵션 최적화:** Rsync 전송 명령에 `--iconv` 플래그를 사용하여 전송 단계에서 NFD ➡️ NFC 변환을 강제합니다.
   ```bash
   rsync -avz --iconv=utf-8-mac,utf-8 /source/path /dest/path
   ```
2. **상태별 물리 디렉토리 격리 (Hand-off):** 같은 폴더 내에서 파일 상태만 바뀌는 경우 덮어쓰기 위험이 크므로 아래와 같이 물리 폴더를 나누어 상태별로 이동(Move)하도록 합니다.
   * `01-raw-analyzed/` : 아톰이 작성한 `status: raw_analyzed` 요약본 보관 (모하비로 단방향 전송)
   * `02-drafting/` : 모하비가 집필 중인 초안 보관 (맥북 로컬 작업 전용)
   * `03-review-pending/` : 작성이 끝나 아우룸에게 넘길 초안 보관 (아톰/아우룸 서버로 역전송)
   * `04-published/` : 최종 PDF/HWP 배포 및 아카이빙 완료된 문서 보관

### 💡 개선안 2. Python 스크립트 단의 NFC 세이프 헬퍼 도입
파일 입출력 시 한글 자모가 깨지는 현상을 방지하기 위해 파일 경로와 파일명을 다루는 파이썬 공통 유틸 함수를 구현하여 코드 내에서 호출합니다.
```python
import unicodedata
import os

def to_nfc_path(path: str) -> str:
    """모든 파일 경로명을 유니코드 NFC 표준으로 정규화합니다."""
    return unicodedata.normalize('NFC', path)

def safe_exists(path: str) -> bool:
    return os.path.exists(to_nfc_path(path))
```
