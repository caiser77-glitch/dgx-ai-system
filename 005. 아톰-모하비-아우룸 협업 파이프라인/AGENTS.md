## 🔌 1. 3단 협업 에이전트 역할 정의 (Agent Roles)

본 협업 파이프라인은 각 인프라 환경의 강점을 극대화하기 위해 아래와 같이 3단계로 분업화되어 유기적으로 상호 작용합니다.

```text
 1단계: 아톰 (Atom)           2단계: 모하비 (Mohave)          3단계: 아우룸 (Aurum)
 [Ubuntu Linux / DGX]      [macOS / MacBook]            [Windows or Server]
 - NAS 실시간 감시 & 추출    - 1차 요약본 RAG 검색        - 보고서 서식 검수
 - vLLM 72B 1차 요약        - 조사 사업 형식 매핑         - 누락 데이터 검증
 - summary.md 생성         - draft.md 보고서 초안 작성   - PDF/한글 정식 배포
```

1. **아톰 (Atom 서버 에이전트):**
   * **환경:** 우분투 리눅스(DGX) 서버 환경.
   * **책무:** 실시간 NAS 감시를 통해 생성되는 원본 문서에서 텍스트를 추출하고, vLLM(Qwen 72B)을 사용해 1차 요약과 카테고리/태그 정보를 추출하여 경량 요약 노트(`*.summary.md`)를 작성합니다.
2. **모하비 (Mohave 맥 에이전트):**
   * **환경:** macOS 로컬 옵시디언 볼트 환경.
   * **책무:** 아톰이 동기화한 `*.summary.md`를 근거로, 사내 RAG 지식베이스에서 과거의 유사 용역 보고서와 매뉴얼을 탐색하여 규격화된 정식 생태조사 보고서 및 용역 초안 문서(`*.draft.md`)를 자동으로 작성합니다.
3. **아우룸 (Aurum 배포 에이전트):**
   * **환경:** 최종 배포 및 텔레그램 게이트웨이 환경.
   * **책무:** 모하비가 작성한 초안(`*.draft.md`)에 대해 데이터 오류나 규격 정밀성을 최종 검수(Review)하고, 검수가 끝난 문서를 HWP/PDF/Docx 정식 포맷으로 변환해 NAS 배포 폴더로 이동시킨 후 완료 소식을 메신저로 전송합니다.

## 🛠️ 2. 파이프라인 트래킹 프로토콜 (File Status Pipeline)

협업 시 문서의 무결성을 유지하고 상태를 추적하기 위해 생성되는 마크다운 노트의 Frontmatter 영역에 아래의 상태(`status`) 메타데이터를 필수적으로 바인딩하고 갱신합니다.

*   `status: raw_analyzed` (아톰 1차 가공 완료)
*   `status: drafting` (모하비가 초안 작성 중)
*   `status: review_pending` (모하비 작성이 완료되어 아우룸에게 검수 의뢰)
*   `status: reviewed` (아우룸의 수치/서식 검증 완료)
*   `status: published` (NAS에 정식 배포 완료)

## 📋 3. 핵심 개발 및 협업 지침 (Rules)

1. **역할 혼동 금지 (SOUL.md 동기화):**
   * 아톰 에이전트는 절대 자신을 모하비(macOS)나 아우룸 에이전트로 혼동하지 마십시오. 아톰의 리소스로는 요약 및 RAG 벡터 색인을 수행하며, 초안 생성이나 HWP 서식 배포는 모하비와 아우룸에 업무를 이관(Delegation)해야 합니다.
2. **NFC 자모 정규화 준수:**
   * macOS(모하비)와 Linux(아톰) 간에 파일 동기화 시 한글 자모분리 현상(NFD)이 발생하므로, 파일 경로를 읽고 쓸 때 무조건 `unicodedata.normalize('NFC', path)` 처리를 하여 정합성 오류를 예방하십시오.
3. **피드백 DB 참조 최우선:**
   * 아톰의 요약 및 1차 분류 시, `/home/caiser77/dgx_workspace/004. 에르메스 NAS 분류기/data/aurum_nas_rules.db`를 조회하여 사용자가 피드백 봇을 통해 승인해 둔 분류 교정 지시가 있는지 확인하고, 존재할 경우 이를 최우선으로 자동 반영하십시오.
4. **결과물 언어:**
   * 에이전트 간의 교환 데이터, 생성되는 문서, 상태 보고서 등 모든 텍스트의 작성은 **한국어**로 정제하여 제공되어야 합니다.
